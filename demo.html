About This Effect

This page demonstrates a responsive video container that changes size when you scroll. The video container has the class name "vd-cnt" as requested.

When the page is loaded, the video container has a width of 1200px. As you scroll down, JavaScript detects the scroll position and expands the video container to 1440px width.

The video is set to auto-play in a loop with muted audio, similar to a banner video.

How It Works

The effect is achieved with a combination of:

  • HTML5 video element with autoplay, loop, and muted attributes
  • CSS for styling and smooth transitions
  • JavaScript to handle the scroll event and video controls
  • Bootstrap for responsive layout

Try resizing your browser window to see how the layout adapts to different screen sizes while maintaining the aspect ratio of the video.

Video Controls

Use the control buttons at the bottom of the video player to:

  • Play/pause the video
  • Mute/unmute the audio
  • Toggle fullscreen mode
